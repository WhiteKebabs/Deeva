# Deeva

## Program Summary
Deeva is a Calendar/Personal Scheduler whose purpose is not only to save and organise plans, but also to allocate and rearrange plans if necessary to give you the most optimal (and appropriate) schedule.

### Bugs
- Events can't go beyond the day
- Doesn't check if repeated events clash in the future
- Deselecting the repeat button doesn't work as intended
- Able to interact with background when in event creation

### Unimplemented Features
- Coloured event buttons
- Zoom out to weekly view
- Event editing interface

- Display available times
- Display available rearrangements
